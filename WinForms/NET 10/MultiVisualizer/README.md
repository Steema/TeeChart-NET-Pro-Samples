# TeeChart for .NET Pro
## Realtime demo dashboard for NET.
 
The demo runs a time based simulation of real-time data entering a system to track several different indicators via different chart and gauge types.<br>

There are two forms in the demo. The first shows how TeeChart's multi axes can be setup:

![screenshot](https://www.steema.com/docs/gallery/charts/assets/charts/dashboard/multiaxischart.png ".NET 10 multi-axis Chart on Winform")

...and this is the dashboard created on the second demo form:

![screenshot](https://www.steema.com/docs/gallery/charts/assets/charts/dashboard/DataHeatVisualiser_charts_winform.png ".NET 10 Dashboard on Winform")
